---

The customer overview displays all customer records in a searchable, filterable list. From here you can add new customers or open an existing customer's detail page.

## Customer list columns

The list displays the following columns for each customer:

| Column | Description |
|--------|-------------|
| **Working name** | The name your team uses to identify this customer |
| **Official name** | Legal name retrieved from VIES or entered manually |
| **VAT number** | The customer's validated VAT registration number |
| **Country** | Country of registration |
| **Payment conditions** | Agreed payment terms (e.g., "30 days net") |
| **Status** | Active or Inactive badge |

## Filtering the customer list

Use the filter bar above the list to narrow down the customers you see. You can combine multiple filters simultaneously.

| Filter | Options |
|--------|---------|
| **Search bar** | Searches by customer name or VAT number |
| **Country** | Dropdown with all countries in your customer base |
| **Status** | Active or Inactive |

> [!tip]
> Type a partial VAT number in the search bar to quickly locate a customer when you only have their tax registration details.


## Actions

### Add a new customer

Click the **Add customer** button in the top-right corner to open the customer creation form. See [[user-guide/customers/creating-customer|Creating a customer]] for the full procedure.

> [!warning]
> Only users with the **Admin** or **Commercial** role can create and edit customers. If you do not see the Add customer button, contact your administrator.


### View customer details

Click any row in the customer list to navigate to the customer detail page. The detail page shows all customer information, linked contacts, and associated contracts.
