---
title: "Activity stream"
description: "Track the latest changes and new records across contracts, offers, and trailers in the ARMS activity stream."
---

The activity stream occupies the right panel of the [dashboard](/user-guide/dashboard/overview). It shows the 20 most recently modified or created records, giving you a quick view of what is happening across the system.

## What the activity stream shows

The stream displays recent activity for three entity types:

- **Contracts** -- New contracts, status changes, and updates
- **Offers** -- New offers, sent offers, and status changes
- **Trailers** -- New trailers added, status changes, and edits

Each entry in the stream follows this format:

```
[Entity icon]  [Name/ID] -- [Action] -- [By user] -- [Relative time]
```

**Example entries:**

| Icon | Entry |
|------|-------|
| Contract | K28 - CompanyX -- Status changed to "In rental" -- by Jan Peeters -- 2 hours ago |
| Offer | OFF-2026-0051 -- Created -- by Marie Dubois -- 4 hours ago |
| Trailer | K15 - 1-ABC-123 -- Inspection added -- by Luc Vermeersch -- yesterday |

## Interacting with the activity stream

Click any entry in the stream to navigate directly to the detail screen for that record. For example:

- Clicking a contract entry opens the contract detail page
- Clicking an offer entry opens the offer detail page
- Clicking a trailer entry opens the trailer detail page in the Fleet module

<Callout kind="info">
  The activity stream shows the last 20 entries. It does not support scrolling to load older entries. For a complete audit trail of changes, use the list views in the respective modules.
</Callout>

## What counts as activity

The stream captures two types of events:

| Event type | Description | Examples |
|------------|-------------|----------|
| **Created** | A new record was added to the system | New contract, new offer, new trailer |
| **Modified** | An existing record was updated | Status change, field edit, document upload |

The stream is sorted by timestamp, with the most recent activity at the top.

<Callout kind="tip">
  Use the activity stream to stay informed about your team's work. If you see a contract status change you were not expecting, click it to review the details immediately.
</Callout>

## Who sees what

All user roles can view the activity stream on the dashboard. The stream shows activity across the entire system -- it is not filtered by the current user's actions.

| Role | Activity stream access |
|------|----------------------|
| Admin | View all activity |
| Commercial | View all activity |
| Accounting | View all activity |
| Fleet manager | View all activity |
| Read-only | View all activity |
