The Notifications page is the full notification inbox for your ARMS account. It provides a centralized, paginated list of all system notifications you have received.

<Callout kind="info">
  This page is accessible to all user roles.
</Callout>

<Callout kind="tip">
  This inbox is separate from the notification widget on the Dashboard. The Dashboard widget shows only the most recent unread notifications, while this page displays your complete notification history.
</Callout>

## Notification list

Notifications are displayed in reverse chronological order with pagination. Each entry shows the notification type, a summary message, the timestamp, and a read/unread indicator.

## Notification types

ARMS generates notifications for the following events:

| Type | Description |
|------|-------------|
| **Inspection expiring** | A trailer inspection is approaching its expiry date |
| **Contract status change** | A contract has moved to a new status (e.g., active, expired, terminated) |
| **Offer response** | A customer has accepted or declined an offer |
| **System alert** | General system notifications such as maintenance windows or import results |

## Interacting with notifications

Each notification links directly to the relevant record. Click a notification to navigate to the associated trailer, contract, offer, or other entity.

To mark a notification as read, click the notification or use the mark-as-read action. Read notifications appear with reduced visual emphasis in the list.
