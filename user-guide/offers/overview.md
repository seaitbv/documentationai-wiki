---
title: "Offers overview"
description: "Browse, filter, and manage all rental offers in ARMS with status tracking and due date monitoring."
---

The offers overview displays all rental offers in a filterable list. From here you can create new offers, monitor due dates, and track offer statuses across the pipeline.

## Offer list columns

The list displays the following columns for each offer:

| Column | Description |
|--------|-------------|
| **Offer ID** | Auto-generated unique identifier |
| **Name** | The offer name you assigned |
| **Customer** | Customer company name |
| **Company** | Atrac or Urbain |
| **Created date** | Date the offer was created |
| **Due date** | Offer expiry date, highlighted in yellow if expired |
| **Status** | Color-coded status badge |
| **Unit price** | Rental unit price |
| **Trailer** | Reference and plate number of the linked trailer, if any |

## Status badges

Each offer displays a color-coded badge indicating its current status:

| Status | Color | Meaning |
|--------|-------|---------|
| Created | Gray | Offer has been drafted but not yet sent |
| Sent | Blue | Offer has been emailed to the customer |
| Accepted | Green | Customer has accepted the offer |
| Refused | Red | Customer has declined the offer |

> [!info]
> When an offer's due date has passed and the status is still Created or Sent, the due date column is highlighted in yellow to draw your attention.


## Filtering the offer list

Use the filter bar above the list to narrow down the offers. You can combine multiple filters simultaneously.

| Filter | Options |
|--------|---------|
| **Status** | Multi-select: Created, Sent, Accepted, Refused |
| **Company** | Atrac or Urbain |
| **Customer** | Customer lookup |
| **Period** | Date range filter on creation date (from -- to) |

> [!tip]
> Filter by status "Sent" and check for yellow-highlighted due dates to identify offers that need follow-up with the customer.


## Actions

### Create a new offer

Click the **Add offer** button in the top-right corner to open the offer creation form. See [[user-guide/offers/creating-offer|Creating an offer]] for the full procedure.

### View offer details

Click any row in the offer list to navigate to the offer detail page, where you can edit the offer, send it via email, or convert it to a contract.
