The Documents tab on the [[user-guide/fleet/trailer-details|trailer detail screen]] lets you attach files to a trailer for reference and record-keeping. You can upload inspection certificates, photos, repair reports, and any other relevant documents.

## Uploading documents

You can upload files in two ways:

- **Drag and drop** - Drag files from your file explorer or email attachment directly onto the upload zone.
- **Browse** - Click the **Browse** button to open your file explorer and select files.

### Supported formats

| Format | Extension |
|--------|-----------|
| PDF | `.pdf` |
| JPEG | `.jpg`, `.jpeg` |
| PNG | `.png` |
| TIFF | `.tif`, `.tiff` |

> [!info]
> The maximum file size per upload is configurable by your administrator. The default limit is **25 MB**.


## Document list

All uploaded documents appear in a list below the upload zone.

| Column | Description |
|--------|-------------|
| **Filename** | The original name of the uploaded file |
| **Type** | Document type selected from a dropdown (e.g., Inspection certificate, Photo, Report) |
| **Upload date** | Date and time when the file was uploaded |
| **Uploaded by** | The user who uploaded the file |
| **Size** | File size in KB or MB |
| **Actions** | Download, Delete, and Preview options |

### Document types

Each document can be assigned a type from a configurable dropdown. This helps organize files and makes them easier to find. Your administrator can add or modify document types through the [[user-guide/administration/parameters|Parameters]] module.

## Uploading a document

### Step 1: Open the Documents tab

Navigate to the trailer's detail screen and select the **Documents** tab.

### Step 2: Upload your file

Drag a file onto the upload zone or click **Browse** to select a file from your computer.

### Step 3: Assign a document type

Select the appropriate document type from the dropdown menu.

### Step 4: Verify the upload

The file appears in the document list with the upload date and your username.


## Document actions

### Download

Click the **Download** button to save a copy of the file to your computer.

### Preview

Click **Preview** to open a document viewer directly in the browser. This works with PDF files and displays them inline without requiring a separate application.

### Delete

Click **Delete** to remove a document from the trailer. A confirmation prompt appears before the file is permanently removed.

> [!warning]
> Deleting a document is permanent and cannot be undone. Make sure you have a copy of the file if you may need it later.
